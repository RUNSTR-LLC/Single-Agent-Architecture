# Skill: Memory + Obsidian Sync

## Goal
Maintain durable project memory with low noise.

## Daily
- Append key events to `memory/daily/YYYY-MM-DD.md`
- Capture decisions, rejections, and blockers

## Weekly
- Distill durable lessons into `memory/LONG_TERM.md`
- Remove stale or duplicate rules

## Logging Rules
- Prefer concise bullets with timestamps
- No secrets in logs
- Link artifacts (PRs, audit files, run proofs)

## Output
- `MEMORY_SYNC_CREATED`
