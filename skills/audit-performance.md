# Skill: Performance Audit

## Goal
Identify high-impact bottlenecks without risky over-optimization.

## Procedure
1. Profile critical user flows.
2. Locate top latency and resource hotspots.
3. Prioritize by user impact and effort.
4. Propose focused fixes with expected gain.
5. Validate with before/after measurements.

## Output
- `PERF_AUDIT_CREATED`
- Ranked bottlenecks + patch recommendations
