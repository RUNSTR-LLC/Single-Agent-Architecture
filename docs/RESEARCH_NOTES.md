# Research Notes (Claims to Verify)

Use this file to pin exact citations used in talks/articles.

## Core claims
- Multi-agent coordination overhead grows super-linearly beyond small team size.
- Single-agent+skills can match task accuracy with lower token/latency cost in many domains.
- Multi-agent failures often stem from context fragmentation and handoff complexity.
- Multi-agent remains useful for truly parallel, low-coupling tasks.
