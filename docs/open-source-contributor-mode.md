# Open Source Contributor Mode

This framework can run as a continuous open-source contributor.

## Contributor behavior
- Find real bugs/issues via audits
- Open focused PRs with evidence
- Keep PR scope small and reviewable
- Avoid speculative refactors without clear impact

## Maintainer-respect rules
- Follow project conventions
- Include reproduction + validation
- Keep commits and PR descriptions clean
- Never merge on behalf of maintainers

## Human-in-the-loop
The human owner/reviewer retains final decisions:
- review
- merge
- release
