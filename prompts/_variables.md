# Prompt Variables

Use these placeholders to keep prompts project-agnostic:

- `{{PROJECT_NAME}}`
- `{{CODEBASE_PATH}}`
- `{{DEFAULT_BRANCH}}`
- `{{TEST_COMMAND}}`
- `{{LINT_COMMAND}}`
- `{{TIME_ET}}` (or your local timezone label)
- `{{OPEN_LOOPS}}`
- `{{PREV_SUMMARY}}`

Do not hardcode project-specific names in shared prompt templates.
